🚀1. 시작하기

로그인

브라우저에서 firstaibot.com에 접속하면 로그인 화면이 나타납니다.

  • 관리자에게 받은 아이디와 비밀번호를 입력하세요.
  • 임시 비밀번호를 받은 경우, 첫 로그인 시 비밀번호 변경이 필요합니다.
  • Guest 계정: 아이디 guest로 로그인 (제한된 기능).

화면 구성

  • 왼쪽 사이드바 — 모드, AI 모델, 페르소나, 전략, 워크스페이스 선택
  • 중앙 채팅 패널 — AI와 대화
  • 오른쪽 출력 패널 — 보고서, 슬라이드, 차트 등 결과물 표시
  • 패널 사이 경계선을 드래그하여 크기를 조절할 수 있습니다.

💬2. 채팅 — AI와 대화하기

하단 입력창에 질문을 입력하고 Enter 또는 전송 버튼을 클릭하세요.

  • AI의 응답은 실시간 스트리밍으로 한 글자씩 표시됩니다.
  • 이전 대화 내용을 기억하여 연속적인 대화가 가능합니다.
  • Shift + Enter로 줄바꿈을 할 수 있습니다.
🎙️ 마이크 버튼을 클릭하면 음성 입력이 가능합니다 (Chrome/Edge 권장).

🤖 자동 라우팅 (Auto Mode)

질문의 복잡도를 자동 분석하여 최적의 모델을 선택합니다:

  • 간단한 질문 (번역, 요약, 계산 등) → 💚 저비용 모델
  • 복잡한 질문 (분석, 전략, 코드, 세무 등) → 🟡🔴 고급 모델
자동 라우팅 덕분에 비용은 절약하면서도 복잡한 질문에는 최고 성능을 받을 수 있습니다.

🎭5. AI 페르소나

페르소나란?

AI에게 특정 역할을 부여하여 전문가처럼 응답하게 합니다:

  • 75개 이상의 기본 페르소나 — CPA, 변호사, 마케터, CEO, 엔지니어 등
  • 사용자 정의 페르소나 — 이름/직업 입력 → AI가 자동으로 성격, 스킬, 응답 스타일 생성

사용 방법

  1. 사이드바 Persona 섹션에서 원하는 페르소나 클릭
  2. 선택한 페르소나가 활성화되면 채팅 시 해당 역할로 응답
  3. + 추가 버튼으로 새 페르소나 생성
페르소나는 대화를 통해 자동 학습합니다. 대화가 쌓일수록 더 정확한 응답을 제공합니다.

🧠8. 장기 기억 시스템

AI가 대화에서 핵심 정보를 자동 추출하여 기억합니다:

  • "나는 CPA입니다" → AI가 이후 대화에서 자동으로 참고
  • "우리 회사는 직원 50명입니다" → 기억에 저장됨

관리 방법

사이드바 🧠 Memory 버튼 클릭:

  • 저장된 기억 목록 확인
  • 수동으로 기억 추가/삭제
  • 전체 삭제 가능
Guest 계정은 기억 기능을 사용할 수 없습니다.

❓15. 자주 묻는 질문

Q: AI 응답 언어를 바꿀 수 있나요?

네! 한국어로 질문하면 한국어로, 영어로 질문하면 영어로 답합니다. "영어로 답해줘" 등의 명시적 요청도 가능합니다.

Q: 대화 내용이 저장되나요?

네, 모든 대화는 클라우드(Supabase)에 자동 저장됩니다. 좌측 대화 목록에서 이전 대화를 다시 불러올 수 있습니다.

Q: 업로드한 파일은 안전한가요?

파일은 AI 분석 목적으로만 사용되며, 세션 종료 시 서버에서 삭제됩니다. 장기 저장을 원하면 워크스페이스에 저장하세요.

Q: 모바일에서도 사용 가능한가요?

네! 반응형 디자인으로 모바일 브라우저에서도 사용 가능합니다. Chrome 또는 Safari를 권장합니다.

Q: Auto 모드에서 어떤 모델이 선택되었는지 알 수 있나요?

AI 응답 상단에 사용된 모델명이 표시됩니다. Owner는 수동으로 모델을 직접 선택할 수도 있습니다.

🚗16. 운전모드 (Driving Mode)

운전모드란?

운전 중 핸즈프리로 AI를 사용할 수 있는 모드입니다. 화면에는 마이크 버튼만 표시되고, AI 응답이 자동으로 음성 출력됩니다.

사용 방법

  1. 사이드바 하단 🚗 운전모드 버튼 클릭
  2. 화면에 큰 🎙️ 마이크 버튼만 표시됨
  3. 마이크 탭 → 말하기 → AI가 자동 응답
  4. AI 응답 후 자동으로 마이크 재시작

음성 중단 및 종료

동작방법
🔊 AI 음성 중단마이크 버튼 탭 1회 → 음성 중단
🎙️ 다시 말하기마이크 버튼 탭 2회 → 마이크 재시작
✕ 운전모드 종료오른쪽 상단 ✕ 종료 버튼 탭
🗣️ 음성으로 종료"운전모드 취소" 또는 "입력모드" 라고 말하기
운전 중에는 음성 중단 + 다시 말하기가 탭 2회이미로 안전합니다.
