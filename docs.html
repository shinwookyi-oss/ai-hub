빠른 시작

from ai_hub import AIHub

hub = AIHub()

# 1. 단일 AI 질문
resp = hub.ask("AI의 미래는?", provider="chatgpt")
print(resp.content)

# 2. 모든 AI 동시 비교
results = hub.ask_all("양자컴퓨터란?")
for r in results:
    print(f"{r.provider}: {r.content[:100]}")

# 3. 사용 가능한 AI 확인
print(hub.available_providers())
# → ['chatgpt', 'gemini', 'azure', 'claude', 'grok']

# 4. 연결 상태 확인
print(hub.status())
# → {'chatgpt': 'Ready', 'gemini': 'Ready', ...}

단일 질문 — ask()

METHOD hub.ask(prompt, provider, system_prompt) → AIResponse
파라미터타입기본값설명
promptstr—질문 내용
providerstr"chatgpt"chatgpt / gemini / azure / claude / grok
system_promptstr""시스템 프롬프트 (역할 지정)

반환: AIResponse — provider, model, content, success, error, elapsed_seconds

resp = hub.ask("요약해줘", provider="claude")
if resp.success:
    print(resp.content)
    print(f"{resp.elapsed_seconds}초 소요")
else:
    print("오류:", resp.error)

비교 모드 — ask_all()

사용 가능한 모든 AI를 동시에 호출하고 결과를 응답 시간 순으로 정렬합니다.

results = hub.ask_all("블록체인의 장단점을 설명해줘")

for r in results:
    print(f"[{r.elapsed_seconds}s] {r.provider} ({r.model})")
    print(r.content)
    print("---")

자동 폴백 — ask_with_fallback()

첫 번째 AI가 실패하면 자동으로 다음 AI로 전환합니다.

resp = hub.ask_with_fallback(
    "안전한 질문",
    priority=["azure", "chatgpt", "gemini"]
)
print(f"{resp.provider}이 답변함")

AI 토론 — debate()

두 AI가 서로 다른 입장에서 토론하고, 세 번째 AI가 심판 역할을 합니다.

result = hub.debate(
    topic="AI가 인류에게 위협인가?",
    rounds=3,
    ai_for="chatgpt",      # 찬성 측
    ai_against="gemini",  # 반대 측
    judge="claude"         # 심판
)

for entry in result["debate_log"]:
    print(f"[Round {entry['round']}] {entry['speaker']}: {entry['content']}")

print("판결:", result["judgment"])
💡 discuss() — 여러 AI가 협력적으로 주제를 논의하는 모드도 있습니다.

페르소나 질문 — ask_as()

# 일론 머스크 관점으로 질문
resp = hub.ask_as(
    "한국 스타트업 생태계를 어떻게 봐?",
    persona="elon_musk",
    provider="chatgpt"
)
print(resp.provider)  # "ChatGPT as Elon Musk"
print(resp.content)

페르소나 토론 — persona_debate()

result = hub.persona_debate(
    topic="전쟁은 정당화될 수 있는가?",
    persona_for="cao_cao",
    persona_against="musashi",
    ai_for="chatgpt",
    ai_against="gemini",
    judge="claude",
    rounds=3
)
print(result["judgment"])

그룹 토론 — persona_discuss()

여러 페르소나가 한 주제를 두고 다 함께 토론합니다.

result = hub.persona_discuss(
    topic="AI 시대 인간의 역할은?",
    persona_keys=["nietzsche", "carl_jung", "elon_musk", "tokugawa"],
    rounds=2,
    callback=lambda name, rnd, resp: print(f"[R{rnd}] {name}: {resp.content[:80]}...")
)
print("종합 결론:", result["synthesis"])

웹앱 모드

모드설명
💬 Chat선택한 AI와 단일 대화 (히스토리 유지)

🧠 Persona Discussion여러 인물이 그룹 토론

Web API 레퍼런스

엔드포인트MethodBody설명
/api/askPOSTprompt, provider, persona단일 질문
/api/comparePOSTprompt모든 AI 비교
/api/debatePOSTtopicAI 토론
/api/discussPOSTtopicAI 토의
/api/bestPOSTquestion최선 답변 선발
/api/persona_debatePOSTtopic, persona_for, persona_against페르소나 토론
/api/persona_discussPOSTtopic, personas[]그룹 토론
/api/uploadPOSTfile (multipart)파일 업로드/분석
